# App Settings • Quick Start

Demonstrates using `NcAppSettings` (Core Foundation) as a global settings singleton.

How to run
1) Create the folder via menu: `Nappy Cat → Create Settings Folder`.
2) Create an app settings asset: `Nappy Cat → Core → Create App Settings Asset`.
3) Add `NcAppSettingsBootstrap` to an empty GameObject, press Play.

What it does
- Reads `NcAppSettings.Instance`, logs values, and applies `reduceMotion` as an example flag.
