# Pooling Quick Start

This sample demonstrates how **NappyCat Pool** can be used for deterministic warmup and zero-allocation reuse.

## Overview

Demonstrates pooling of reusable projectiles with zero allocations after warmup.

## Instructions

1. Import the sample into your project.
2. Open the included scene and press Play.
3. Inspect the console output to confirm no allocations are reported after warmup.
