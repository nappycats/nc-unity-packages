Overview

NappyCat Tween is a delegate-driven tween engine optimised for runtime gameplay and tooling. All tweens and sequences are pooled, giving you predictable zero-GC behaviour after setup. The API surface is deliberately small and mirrors the simplicity of Unity’s own scripting reference.

Features

  • Typed `To` helpers for floats, vectors, quaternions, and colours.
  • Pooled sequences with serial (`Then`) and parallel (`Also`) composition.
  • Manual ticking support for Timeline and editor previews.
  • Group management for coordinated pause/kill operations.
  • Extension methods for common Unity components.

Installation

  1. Copy the com.NappyCat.Tween folder into your project’s Packages/ directory.
  2. Open Unity and go to Window → Package Manager → + → Add package from disk….
  3. Select Packages/com.NappyCat.Tween/package.json.
  4. (Optional) Import the “Samples” package sample from the Package Manager window.

Quick Start

Create a simple tween using the `NcTween.To` helper:

using NappyCat.Tween;
using UnityEngine;

public sealed class HoverPulse : MonoBehaviour
{
    TweenHandle _handle;

    void OnEnable()
    {
        _handle = NcTween.To(
            () => transform.position,
            p => transform.position = p,
            transform.position + Vector3.up * 0.5f,
            new NcTweenOptions(duration: 0.5f, ease: NcEase.OutQuad));

        NcTween.OnComplete(_handle, () => Debug.Log("Hover finished"));
    }

    void OnDisable()
    {
        NcTween.Kill(_handle);
    }
}

Concepts

Tween handles

Every call to NcTween.To returns a TweenHandle. Use it to pause, kill, or attach callbacks. Handles are tiny structs containing only an ID, so you can store them freely.

Options & Events

NcTweenOptions captures timing, easing, looping, clock source, and group ID. NcTweenEvents provides optional callbacks fired at key points in the tween lifecycle.

Runner lifecycle

The first tween automatically spawns NcTweenRunner on a hidden ~NcTweenRunner GameObject. Set persistentMode to control whether it persists across scenes.

Manual ticking

Switch a tween to NcClock.Manual when you want deterministic stepping:

var options = new NcTweenOptions(1f, clock: NcClock.Manual);
var handle = NcTween.To(() => progress, v => progress = v, 1f, options);

// timeline update
NcTween.ManualTick(deltaTime);

Sequences

Use NcSequence to compose serial and parallel steps. Sequences are pooled and allocation-free after construction.

API Highlights

NcTween.To

Typed factories for float / Vector2 / Vector3 / Vector4 / Quaternion / Color. Accept NcTweenOptions and optional NcTweenEvents.

NcTween.Kill / Pause

Stop or pause a single tween or entire group.

Extension methods

Convenience helpers such as transform.NcMoveTo, canvasGroup.NcAlphaTo, image.NcFillTo, spriteRenderer.NcColorTo.

NcSequence

Fluent composition with Then, Also, and OnComplete. Works seamlessly with core tweens.

NcSequence (class)#

Lightweight sequence builder that orchestrates serial (Then) and parallel (Also) tween steps.

Methods

  • Sequences are reusable when built entirely from pooled tweens.

NcTweenRunner (MonoBehaviour)#

Singleton runner component responsible for calling NcTween.Tick every frame. Spawned automatically on first tween creation.

Behaviour

  • The runner lives on a hidden GameObject named ~NcTweenRunner.
  • Set persistentMode before play if you want scene-local behaviour.
  • Disabling the component allows you to manually drive NcTween.Tick from your own update loop.
