# NappyCat • Extensions

Zero‑alloc helpers for math, collections, and Unity types.

Includes
- NcMath: Smooth01, Remap, NcRoundToNearestHalf, NcRandom*, NcRotatePointAroundPivot*, NcAngle*, NcProjectPointOnLine
- NcMathExtensions: Vector2/3 NcWithX/Y/Z, NcRotate, NcGetClosest; float NcNormalizeAngle/NcRoundDown
- NcUnityExtensions: NcTransformBounds, NcTryGetComponent<T>, NcSetLeft/Right/Top/Bottom (RectTransform), NcWorldSpaceWidth/Height (Camera)
- NcCollectionsExtensions: List<T>.NcForEachNoAlloc(Action<T>), NcShuffle, NcRandomItem, NcRemoveRandom
- NcArrayExtensions: T[].NcRandomValue(), NcShuffle()
- NcDictionaryExtensions: NcTryAdd, NcAddOrUpdate, NcTryRemove, NcKeyByValue
- NcTimeExtensions: float.NcToHumanSeconds()
- NcStringExtensions: NcIsNullOrWhiteSpace()
