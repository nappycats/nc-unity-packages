# NappyCat • Extensions

<div style="background:linear-gradient(135deg,#0d1117,#12202b);border:1px solid #58a6ff;border-radius:10px;padding:12px;color:#c9d1d9;">
<strong style="color:#79c0ff">Sharper primitives.</strong> Zero‑alloc helpers for math, collections, Unity types, and utility glue.
</div>

Includes
- <span style="color:#d2a8ff">Math</span>: `NcMath` (Smooth01, Remap, NcRoundToNearestHalf, NcRandom*, NcRotatePointAroundPivot*, NcAngle*, NcProjectPointOnLine)
- <span style="color:#79c0ff">Math Extensions</span>: `NcWithX/Y/Z`, `NcRotate`, `NcGetClosest`, `NcNormalizeAngle`, `NcRoundDown`
- <span style="color:#56d364">Unity</span>: `NcTransformBounds`, `NcTryGetComponent<T>`, `NcSetLeft/Right/Top/Bottom` (RectTransform), `NcWorldSpaceWidth/Height` (Camera)
- <span style="color:#ffa657">Collections</span>: `List<T>.NcForEachNoAlloc`, `NcShuffle`, `NcRandomItem`, `NcRemoveRandom`
- Arrays: `NcRandomValue`, `NcShuffle`; Dictionaries: `NcTryAdd`, `NcAddOrUpdate`, `NcTryRemove`, `NcKeyByValue`
- Utility: `NcTimeExtensions.NcToHumanSeconds()`, `NcStringExtensions.NcIsNullOrWhiteSpace()`
